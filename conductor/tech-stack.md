# Tech Stack: Lerncycle AI Learning App

## Core Technologies
*   **Languages:**
    *   **TypeScript/JavaScript:** Primary language for the web application (Next.js) and frontend development.
    *   **Python:** Used for AI-specific services and advanced data processing, leveraging its rich ecosystem of AI/ML libraries.

## Frontend
*   **Next.js (React):** A powerful React framework for building robust, scalable, and SEO-friendly web applications.
*   **Shadcn UI:** A collection of clean, highly customizable UI components for a minimalist, Notion-style aesthetic.
*   **Tailwind CSS:** For efficient, utility-first styling and a responsive user interface.

## Backend & Database
*   **Supabase:** A comprehensive, open-source Backend-as-a-Service (BaaS) providing a robust PostgreSQL database, built-in authentication, and real-time capabilities.
*   **PostgreSQL:** The underlying relational database for managing semesters, subjects, weeks, and user-uploaded data.

## AI & Services
*   **OpenAI (GPT-4):** The primary engine for the AI chatbot, Feynman technique sessions, and AI-assisted flashcard generation.
*   **Supabase Storage:** For secure and scalable storage of user-uploaded study materials (PDFs).

## Deployment & DevOps
*   **Vercel:** Preferred hosting platform for seamless Next.js deployments and optimized performance.
